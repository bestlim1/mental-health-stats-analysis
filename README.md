# 정신질환 통계 데이터 분석 (2018~2022)

서울시 시군구별 정신질환 환자 데이터를 분석하여 지역·연령·질환별 패턴을 도출한 프로젝트입니다.

## 주요 분석 결과

- **지역별 분포**: 우울증 환자수가 가장 많은 지역은 강남구·서초구·송파구. 단순 인구 비례가 아닌 경제·교육·의료 접근성 등 사회적 요인과 연관됨을 확인.
- **질환별 추이**: 2018~2022년 사이 6가지 정신질환 모두 환자수 증가. 특히 ADHD는 4년 만에 약 2.5배 증가 (60,909명 → 149,272명).
- **연령별 입내원일수**: 40~60대 중년층의 입내원일수가 가장 많으며, 20~30대(MZ세대)의 상승폭이 두드러짐.
